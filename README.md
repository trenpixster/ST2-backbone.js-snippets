##Trenpixster's Sublime Code Snippets

```
git clone git://github.com/trenpixster/ST2-backbone.js-snippets.git
cd ST2-backbone.js-snippets
```
Then copy the `*.sublime-snippet` to your Sublime Text 2 data folder.
